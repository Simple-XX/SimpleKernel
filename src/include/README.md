# src/include/

## 文件说明

- kernel.h
    声明了内核要用到的函数与变量。

- tty.h
    命令行输出。

- vga.h
    显示相关的函数。
    
- port.h
    端口输入输出。
    
- debug.h
    调试函数
    
- libc
    基础库，包括字符串、输出、类型定义等

- mm/
    内存相关

- pic.hpp

    pic 芯片的初始化与刷新


- multiboot.h

    多重引导规范相关结构

- elf.h

    elf 文件格式相关结构
